# DCEasyRec
简单好用且功能强大的屏幕录制软件



功能：

* 支持全屏录制（目前区域录制和窗口录制功能不可用）
* 支持调整分辨率
* 支持系统声音/麦克风声音组合
* 支持**自动降噪**
* 支持摄像头
* 支持**自动美颜**
* 支持倒计时设置
* 支持鼠标轨迹等
* 支持水印设置
* 支持录制文件位置设置
* 支持录制文件管理（列表，播放，删除等）
* 支持**画笔工具**
* 支持快捷键设置
